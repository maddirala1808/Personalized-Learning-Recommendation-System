# Personalized-Learning-Recommendation-System
This project focuses on building a Personalized Learning Recommendation System that analyzes learner profiles, skills, interests, and learning goals to recommend suitable online courses. Using the Online Courses Across Platforms dataset from Kaggle, the system processes course information, performs feature engineering, and applies machine learning recommendation algorithms to generate accurate and personalized course suggestions. The project demonstrates data preprocessing, recommendation engine development, model evaluation, and performance analysis using metrics such as Precision@K, Recall@K, F1-Score@K, MAP@K, and NDCG@K.
